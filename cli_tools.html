Command Line Tools¶

dripline_agent¶

usage: dripline_agent [-h] [-c configuration file] [-b AMQP broker address]
                      verb target [value [value ...]]

dripline_agent provides basic a CLI to the dripline system.

positional arguments:
  verb
  target
  value

optional arguments:
  -h, --help            show this help message and exit
  -c configuration file, --config configuration file
                        full path to a dripline YAML configuration file.
  -b AMQP broker address, --broker AMQP broker address
                        the address of the AMQP broker dripline should use

start_node¶

usage: start_node [-h] [-c configuration file]

Start a dripline node.

optional arguments:
  -h, --help            show this help message and exit
  -c configuration file, --config configuration file
                        full path to a dripline YAML configuration file.

message_monitor¶

usage: message_monitor [-h] [-a amqp server host address] [-e exchange name]
                       [-k [binding keys [binding keys ...]]]

Print messages from an exchange.

optional arguments:
  -h, --help            show this help message and exit
  -a amqp server host address, --amqp amqp server host address
                        full network name of amqp host
  -e exchange name, --exchange exchange name
                        amqp name of the exchange to monitor
  -k [binding keys [binding keys ...]], --keys [binding keys [binding keys ...]]
                        amqp binding keys to follow
